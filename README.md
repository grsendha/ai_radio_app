## Installation

To install this project run

Step 1: 
```bash
  git clone https://https://github.com/grsendha/ai_radio_app
```
Step 2:
```bash
  flutter pub get 
```

## Library Used

- velocity_x
- audioplayers
- google_fonts
- alan_voice
